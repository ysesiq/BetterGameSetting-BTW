# Better Game Setting for BTW

### v1.3.0
* 适配了3.0.0 Release
* 优化了按键绑定，现在支持重置模组的按键
* 优化了按键绑定界面部分按钮的位置
* 加入了1.19.4+的创建世界界面
  * 移植并大幅修改自 [CreateWorldUI](https://github.com/song682/CreateWorldUI)
  * 加入了游戏规则设置界面，并且使修改的游戏规则能正常生效
* 完善了声音设置界面，但目前对部分音效无法调整
* 资源包界面加入了搜索框，并加入了点击框选效果
* 缩小了输入框的尺寸，使其大小与同尺寸的按钮一致
* 输入框在聚焦时边框颜色变浅
* 加入了按钮长文本滚动效果
* 语言界面加入了搜索框，并且支持使用语言代码搜索
* 优化了视频界面提示框逻辑
* 优化了本模组新增的列表界面的渲染层次顺序
* 加入了不兼容的资源包使用确认窗口
* 加入了`IButton`和`ITextField`用于快速实现修改位置/大小，加入输入框提示，绘制滚动文字
* 优化了部分软件包、类名和代码
* 修复了切换语言时，强制Unicode字体按钮文本不改变的问题
* 内置了Fabric-ASM模组

## 1.1.0
* 移植自BetterGameSetting（MITE）
* 因为Mixin Extra与EnumExtends缺失，暂时不移植声音设置方面内容
* 可以选择多个资源包
* 更好的调整渲染距离和Gui缩放
* 自定义Fps最大值
* 更低的Fov
* 更好的控制设置（注意：目前不支持重置mod的按键绑定）
